# 21차시 데이터의 대기 공간: 스택

스택: 데이터의 삽입과 삭제가 한쪽에서만 일어나는 구조(LIFO: Last In First Out)

PUSH: 스택 삽입 + 스택이 가득 찼는데 한번 더 PUSH하면 오류가 발생한다.

POP: 스택 삭제 + 스택에 아무것도 없는데 POP하면 오류가 발생한다.

연결 리스트로도 구현 가능하다.

## 문제

1. 스택에 아무것도 없는데 POP연산을 하면 어떤 일이 발생하는가?
2. 스택은 어떤 구조를 나타내는지 다음 중 하나를 고르시오
   1. LILO
   2. LIFO
   3. FILO
   4. FIFO
