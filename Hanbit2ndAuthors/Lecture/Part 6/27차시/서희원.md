# 27차시 정렬 알고리즘: 삽입 정렬

아직 정렬되지 않은 임의의 데이터를 이미 정렬된 부분의 적절한 위치에 삽입해가며 정렬하는 방식이다.

```python
arr = input()
n = len(arr)
now = 0
for i in range(1, n):
	now = i
	for j in range(i):
		if arr[now] < arr[j]:
			arr[now], arr[j] = arr[j], arr[now]
```

이 방식은 1부터 n까지 1씩 늘어나는 숫자들을 다 더하면 수행시간이 나온다.(약 n^2이라 본다.)
