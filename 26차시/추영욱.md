# 정렬알고리즘: 선택정렬

## 선택정렬
- 가장 작은 것을 선택해서 제일 앞으로 보내는 알고리즘

### 정렬을 위한 변수들
- i : 정렬할 위치의 인덱스
- j : 인덱스 i+1부터 시작하는 각 데이터의 인덱스
- m : 가장 작은 데이터의 인덱스
- temp : 데이터 교환을 위한 임시 공간


### 선택정렬 알고리즘

```java
algorithm selection(int ds[], int n) {
	for i = 0 to n-2
		m = i
		for j = i+1 to n-1
			if ds[j] < ds[m]
				m = j
			end if
		next j

		if i != m
			temp = ds[m]
			ds[m] = ds [j]
			ds[j] = temp
		end if
	next i
}

```

## 예제 문제
1. 선택정렬의 뜻을 설명하시오.
2. 선택정렬을 하기 위해 변수가 어떻게 필요한가?

## 정답
