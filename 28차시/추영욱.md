# 정렬알고리즘: 버블알고리즘

## 버블정렬알고리즘
- 서로 이웃한 데이터를 비교하여 큰 데이터를 뒤로 보내는 과정을 반복하며 정렬하는 알고리즘


```
algorithm bubble(int ds[], int n) {
	for i = 1 to n-1
		for j = 1 to n-i
			if ds[j-1] > ds[j]
				temp = ds[j-1]
				ds[j-1] = ds[j]
				ds[j] = temp
			end if
		next j
	next i
}
```
