# 프로그램의 실행 단위, 프로세스

## 프로세스의 개념
하드디스크에 저장된 것이 프로그램, 램에 올라가서 실행되는 것이 프로세스

## 프로세스 제어 블록(PCB)
프로세스에 대한 정보를 담고 있는 것
```
생성 -> 준비 <-> 실행 -> 종료
		\__대기__/
```
준비에서 실행으로 넘어가는 것을 디스패치 라고 함
