# 더하기 회로와 기억 회로

## 가산기

| A | B | C | S |
| - | - | - | - |
| 0 | 0 | 0 | 0 |
| 0 | 1 | 0 | 1 |
| 1 | 0 | 0 | 1 |
| 1 | 1 | 1 | 0 |

- $ S = A \oplus B = \overline{A}B + B\overline{A} $
- $ C = AB $


## 플립플롭

- 기억장치의 상태를 파악하기 위한 회로

| $\overline{S}$ | $\overline{R}$ | 상태 |
| -------------- | -------------- | ---- |
| 0              | 0              | 불능 |
| 1              | 0              | 1 |
| 0              | 1              | 0 |
| 1              | 1              | 불변 |


## 예제 문제
1. 플립플롭은 (   )의 (   )를 파악하기 위한 회로이다.
2. $\overline{S}$ 가 0이고, $\overline{R}$ 이 0일 때, 플립플록의 상태는 무엇인가?

## 정답
1. 기억장치의 상태
2. 불능
