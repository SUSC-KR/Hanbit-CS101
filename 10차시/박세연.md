# **10차시\_더하기 회로와 기억 회로**

### 반가산기

- S (Sum) = A ⨁ B = $\overline{A}B + A\overline{B}$ → **XOR**
- C(Carry) = AB → **AND**

### **Filp-Flop**

- 1 bit의 이진 데이터를 저장할 수 있는 기억 장치

### SR Latch

- 가장 기본적인 기억 회로
- **S** : Set, **R** : Reset
- Q’ 는 항상 Q와 반대되는 값을 의미하므로 Q와 Q’ 가 항상 같은 값이 나오는

| $\overline{S}$ | $\overline{R}$ |
| -------------- | -------------- |
| 0              | 0              |
| 1              | 1              |

은 SR Latch에서 사용하지 않음
