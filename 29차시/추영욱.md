# 탐색 알고리즘

## 선형 탐색
- 데이터 집합에서 처음부터 순차적으로 탐색하는 방법

```
algorithm linear(int ds[], int n) {
	input key
	for i = 0 to n-1
		if key == ds[i]
			return i
		end if
	next i
}
```

### 선형탐색의 한계
- 데이터의 크기가 클 경우 매우 비효율적이고, 최악의 경우 모든 요소를 확인해야 함


## 이진 탐색
- 정렬된 데이터 집합을 반으로 쪼개면서 탐색하는 알고리즘

```
algorithm binary (int ds[], int n) {
	low = 0
	high = n-1
	input key
	while low <= high
		mid = (low + high) / 2
		if key == ds[mid]
			return mid
		else if key < ds[mid]
			high = mid -1
		else
			low = mid + 1
		end if
	end while
}

```
