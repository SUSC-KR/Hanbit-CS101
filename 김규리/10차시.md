## 가산기

* 반가산기
```
  0    0 0    0 1      1
+ 0    + 1    0 0      1 
----------------------------
0 0    0 1    0 1    1 0 < 올림수 발생 
``` 

|AB|C|S|
|:---:|:---:|:---:|
|00|0|0|
|01|0|1|
|10|0|1|
|11|1|0|

* S = XOR 연산 = A'B+AB'
* C = AND 연산 = AB


## 플립플롭
1비트의 정보를 저장하는 회로
피드백을 이용하여 Q, Q'를 출력
불능 / 1 / 0 / 불변 상태가 됨
  
* SR 플립플롭
    
|S'|R'|상태|
|:---:|:---:|:---:|
|0|0|불능|
|0|1|1|
|1|0|0|
|1|1|불변|
|1|1|1|0|
