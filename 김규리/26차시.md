## 선택 정렬 : 가장 작은 데이터를 가장 앞의 데이터와 교환
* 데이터를 리스트에 저장 
* 값을 하나씩 증가하면서 

## 코드

```
#define SWAP(x, y, t) ( (t)=(x), (x)=(y), (y)=(t) )

void selection (int list[], int n){
	int i, j, least, temp;
	for(i=0; i<n-1; i++) {
		least = i;
		for(j=i+1; j<n; j++) // 최솟값 탐색
			if(list[j]<list[least]) least = j;
		SWAP(list[i], list[least], temp);
	}
}
```
